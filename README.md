# 🗂 Logical Volume Management (LVM) Lab

### 🏗 Lab Scenario: Teacher / Student Storage Architecture

## Project Overview

This lab was built as part of my RHCSA preparation and to strengthen real world Linux storage administration skills.

This project demonstrates hands-on implementation of Logical Volume Management (LVM) in a Red Hat Enterprise Linux environment.

The goal was to simulate a school storage architecture while learning logical volume managment using:

- Disk Partitioning
   
- Physical Volumes 
  
- Volume Groups 

- Logical Volumes 

- Persistent Mounting

- Filesystem Creation

- Storage Segmentation by Role
  
- Volume and Filesystem Resizing

After I finished creating this entire architecture from scratch I decided to utilize ai to create a bash script to do the entire thing for me. I spun up a second VM and ran the script. It created it within seconds. I did have to adjust the parameters of the second partition from a max of 5gb to 100% of left over disk space but other than that the script was perfect. That just goes to show how much more efficient we can be if we learn to utilize both the intelligence of ai and the automation of scripting!
